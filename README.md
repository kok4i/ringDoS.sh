# ringDoS.sh
Interactive bash script that searches for nearby Ring devices and allows you to 'DoS' them using aireplay-ng to send disassocation packets

## Required Packages
*aircrack-ng*  
*iwconfig*  

Debian Install
```
apt install aircrack-ng
```
```
apt install iwconfig
```
  
Arch Linux Install
```
pacman -S aircrack-ng
```
```
pacman -S iwconfig
```
